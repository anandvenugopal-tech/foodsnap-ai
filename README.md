# FoodSnap AI  
### AI-Powered Food Recognition & Nutrition Analysis

FoodSnap AI is a deep learning–based application that identifies food items from images and provides detailed nutritional information such as calories, protein, fat, and carbohydrates.  
It uses an EfficientNet model trained on a 20-category food dataset and integrates with a MySQL database for accurate nutrition lookup.  
The project features a modern Streamlit UI with clean white cards for a smooth user experience.

---

## Tech Stack

- **Python 3.9+**
- **TensorFlow / Keras (EfficientNet)**
- **Streamlit**
- **PIL (Pillow)**
- **NumPy / pandas**
- **MySQL Database**

---

## How It Works

1. User uploads a food image
2. Image is preprocessed and passed into EfficientNet
3. Model returns the predicted food category
4. Nutrition info is fetched from MySQL
5. Results displayed in elegant white cards on Streamlit
